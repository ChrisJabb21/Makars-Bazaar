# JS Ecommerce app 

Ecommerce shopping web app made with JavaScript, NodeJS, and MongoDB backend.

## Technologies used
---
- HTML/CSS
- JavaScript 
- NodeJS
- MongoDB

## Project Structure 
Directory layout displayed with the ``` tree ``` in Linux and/or Windows command prompt in root project directory.

```

        │   └─── ProductScreen.js
        ├─── app.js
        ├─── data.js
        ├─── index.html
        ├─── styles.css
```
